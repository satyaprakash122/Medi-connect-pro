# Medi-connect-pro

MediConnect Pro is an AI-powered telemedicine and healthcare management platform.

The platform connects patients, doctors, hospitals, vendors, and service engineers through a unified ecosystem.

## Features

### Telemedicine

* Patient Registration
* Doctor Registration
* Appointment Booking
* Video Consultation
* Chat Messaging

### Medical Equipment Marketplace

* B2B Equipment Ordering
* Vendor Management
* Service Booking
* Equipment Maintenance Tracking

### AI Features

* AI Symptom Checker
* AI Prescription Summary
* AI Healthcare Assistant

## Tech Stack

### Backend

* Java
* Spring Boot
* Spring Security
* JWT Authentication
* MySQL
* JPA / Hibernate

### Frontend

* React
* Tailwind CSS
